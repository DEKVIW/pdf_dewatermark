# 产品命名与图标

## 产品名

| 项 | 内容 |
| --- | --- |
| 中文名 | **净页** |
| 英文名 | **JingYe** |
| 全称 | 净页 JingYe |
| 窗口标题 | 净页 — PDF 清理工作台 |
| 可执行文件 | `JingYe.exe` |
| 版本 | 以 `src/pdf_dewatermark/gui/branding.py` 的 `APP_VERSION` 为准（当前 0.2.2） |

**命名含义**：「净」清理杂质色，「页」面向 PDF 页面。

## 图标资源

| 文件 | 用途 |
| --- | --- |
| `src/pdf_dewatermark/gui/resources/app.ico` | 窗口 / 任务栏 |
| `src/pdf_dewatermark/gui/resources/app.png` | 备用 |
| `packaging/app.ico` | PyInstaller 嵌入 |

## 应用内图标（Fluent）

统一入口：`src/pdf_dewatermark/gui/icons.py`  

新增按钮/导航时先改 `icons.py`，再在页面引用。完整导航与工具映射见 **`07-GUI开发说明.md` §7**。
