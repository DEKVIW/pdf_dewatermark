# 现状与能力清单

> 基准：仓库改造前脚本集合（已迁入 `legacy/`）  
> 日期：2026-07-15

## 1. 改造前文件

| 路径 | 职责 |
| --- | --- |
| `legacy/pdf去除正文水印3.py` | Tk 主 GUI：颜色点选 / 色阶阈值、预览、单页/范围/全量、灰度转换 |
| `legacy/灰阶去水印/01 彩色PDF转灰色PDF.py` | 多进程灰度渲染 |
| `legacy/灰阶去水印/02 色阶去水印.py` | 命令行色阶去水印（硬编码路径） |
| `legacy/pdf页眉页脚加矩形.py` | 手动画矩形并盖白块 |
| `legacy/计算图片宽高.py` | 查看 PDF 内嵌图尺寸（辅助） |

## 2. 已有能力

- 交互式点选水印色、自定义背景色
- 颜色欧氏距离容差、对比度、DPI/分辨率因子
- 预览 / 当前页 / 页范围 / 全部导出
- 灰度转换（多进程批处理思路）
- 页眉页脚区域遮盖（独立工具）

## 3. 已知问题

| 类别 | 问题 |
| --- | --- |
| 结构 | 单文件巨型 GUI，UI 与算法耦合；脚本逻辑重复 |
| 效果 | 阈值掩码按通道 OR，易误伤；硬填背景边缘脏 |
| 一致性 | 预览用显示图、导出用另一套 DPI |
| 产物 | 全程光栅化 → 图片型 PDF，丢失可选中文字 |
| 工程 | 无包结构、无依赖清单、无测试、无打包方案 |
| 体验 | Tk 界面；多工具割裂；参数无法持久化 |

## 4. 设计边界（本项目默认）

- **主场景**：课件 / 讲义 / 扫描或渲染后的「浅色半透明水印」PDF  
- **不承诺**：完整保留文字可选中/可搜索（光栅路线的固有代价；文档中明确告知用户）  
- **矢量对象水印**（批注、OCG）：二期可选，不阻塞首版
