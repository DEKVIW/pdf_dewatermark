# 效果优化方案

## 1. 原则

1. **预览 ≡ 导出**：同一函数处理单页图像；仅分辨率/页数不同  
2. **先准后美**：掩码准确优先于锐化装饰  
3. **可解释参数**：容差、填充、形态学都有明确语义  

## 2. 相对 legacy 的改进

| 项 | 旧行为 | 新行为 |
| --- | --- | --- |
| 颜色匹配 | 色阶模式通道 OR，易误伤 | 统一 **RGB 欧氏距离**；可选 **加权 Lab 近似** |
| 多点取样 | 单点 | 支持多次点击，颜色取中位数，抗噪声 |
| 掩码 | 无后处理 | 开运算去噪 + 闭运算补洞（PIL Min/MaxFilter） |
| 填充 | 硬替换背景色 | `solid` 硬填 / `soft` 边界羽化 / `neighbor` 邻域均值 |
| 预览 | 用屏幕显示图 | 用处理 DPI 渲染再缩小显示 |
| 对比度 | 全局 enhance | 默认保守；可关；避免「去水印变成过曝」 |
| 输出 | 临时文件路径 | 优先内存流 `insert_image(stream=…)` |

## 3. 方法定义

### 3.1 `distance`（默认，替代原 color_pick）

像素与目标色欧氏距离 `< tolerance` → 掩码。

### 3.2 `distance_lab`（增强）

将 RGB 转到近似 Lab 后再算距离，对「浅灰 / 浅青」类水印更稳。

### 3.3 区域遮盖 `region_cover`

不依赖颜色，矩形区域内直接盖背景色（整合页眉页脚工具）。

### 3.4 灰度 `grayscale`

渲灰度图再嵌回；作为「色阶类水印」的预处理选项，可从 GUI 一键跳转。

## 4. 推荐默认参数

| 参数 | 默认 | 说明 |
| --- | --- | --- |
| method | `distance` | 通用 |
| tolerance | 28 | 浅灰水印常 20–40 |
| fill_mode | `soft` | 边缘更自然 |
| morph_open | 1 | 去孤立噪点 |
| morph_close | 2 | 补水印空洞 |
| dpi | 200 | 预览/速度平衡；导出可 250–300 |
| contrast | 1.0 | 默认不拉对比度 |
| sharpen | off | 需要时再开 |

## 5. 明确不做（首版）

- 深度学习去水印  
- 保证文字可选中  
- 自动检测水印颜色（可二期：直方图尖峰）  

## 6. 验收方式

用同一份样例 PDF：

1. 水印不可见或明显变淡  
2. 正文笔画不断裂、不大量缺字  
3. 预览与导出观感一致  
4. 50 页文档在中档 PC 可接受时间内完成（目标：≤ 数分钟级，视 DPI）
