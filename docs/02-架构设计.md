## 1. 产品定位

| 项 | 决策 |
| --- | --- |
| 中文名 | **净页** |
| 英文名 | **JingYe** |
| 包名 | `pdf_dewatermark` |

## 2. 目录结构

```text
PDF去水印/
├── docs/
├── legacy/                       # 旧 Tk 脚本归档

│   ├── core/                     # 选色算法、掩码、填充、灰度、区域
│   ├── pdf/                      # 渲染、导出
│   ├── gui/                      # 见 07-GUI开发说明.md
│   ├── models.py
│   ├── processor.py
│   ├── paths.py
│   └── cli.py
├── packaging/
├── scripts/build_gui_onedir.ps1
├── dist/JingYe/                  # 最新绿色包
├── dist/releases/                # 版本归档
├── pyproject.toml
└── requirements.txt
```

## 3. 分层原则

```text
GUI / CLI  →  processor  →  core + pdf
```

- **core**：numpy / Pillow，可单测  
- **pdf**：fitz 读写  
- **processor**：编排预览/导出  
- **gui**：只调 API，不复制算法  

## 4. GUI 信息架构（摘要）

```text
壳层：菜单 + 应用工具栏（打开/文件名/输出）+ 状态栏
左侧导航：选色替换 | 区域遮盖 | 灰度转换 | 批量处理
选色页：空状态 | 连续滚动 + 参数抽屉 + 文档工具栏
```

详细布局、取色、懒渲染见 `07-GUI开发说明.md`。

## 5. 入口

| 入口 | 说明 |
| --- | --- |
