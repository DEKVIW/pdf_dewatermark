净页 JingYe — 使用说明
======================

版本：见同目录 VERSION.txt

一、启动（绿色包 / 解压即用）
  1. 解压后不要只拷贝 JingYe.exe，请保留整个文件夹（含 _internal）
  2. 双击 JingYe.exe 启动
  3. 首次运行可在「文件」菜单打开 PDF，或拖入窗口

二、功能简介
  1. 选色替换
     - 打开 PDF，用吸管单击水印/杂质色（目标色）
     - 可「添加一组」处理多种颜色，一次导出
     - 右侧「参数」可调容差、对比度
     - 导出预设：均衡（推荐）/ 高清 / 小体积（默认 JPEG，体积更小）
     - 扫描页默认不超过原图分辨率，避免放大发糊、体积暴涨
     - 滚轮滚动 · Ctrl+滚轮缩放 · 空格/中键拖拽
     - 「原图 / 效果」切换预览；处理当前页 / 页范围 / 全部

  2. 区域遮盖
     - 拖拽矩形盖住页眉、页脚等
     - 「应用到…」：全部页 / 奇数页 / 偶数页 / 从当前页隔页 / 自定义页码
       （适合隔页水印、集数页等，不必每页重画）

  3. 灰度转换
     - 彩色 PDF 转灰度（输出为图片型 PDF）

  4. 组合处理
     - 同一 PDF 只需顶栏打开一次（全局共用）
     - 在选色/区域页配好参数后，于本页勾选步骤、调顺序，一次导出
     - 各模块仍可单独导出

  5. 批量处理
     - 先在选色替换调好颜色组
     - 点「从选色页同步」再批量跑多个文件

三、输出与结果说明
  - 默认输出目录：程序目录下的 output 文件夹
  - 也可在工具栏更改输出目录
  - 结果一般为「图片型 PDF」，通常无法再选中文字
  - 请仅处理你有权处理的文档

四、预览与导出
  - 窗口预览按屏幕清晰度渲染，便于取色与对比
  - 导出默认「均衡」：JPEG + 约 200 DPI，清晰与体积兼顾
  - 需要极致清晰可选「高清」或自定义 PNG；需要更小文件选「小体积」
  - 处理效果以导出 PDF 为准

五、目录说明（请勿随意删除）
  JingYe.exe      主程序
  _internal/      运行库（必须与 exe 同目录保留）
  VERSION.txt     版本号
  使用说明.txt    本说明
  data/           偏好设置等
  output/         默认输出
  logs/           日志（若有）

六、常见问题
  Q：双击没反应？
  A：确认 _internal 文件夹完整；可尝试以管理员身份运行；杀毒软件可能误拦首次启动。

  Q：预览清晰但导出很大/很慢？
  A：导出 DPI 越高越清晰、越占空间。可在参数中适当降低 DPI 再试。

  Q：文字选不中了？
  A：选色/灰度等流程会光栅化页面，这是预期结果。

七、安装版（可选）
  若你拿到的是 JingYe-Setup-x.y.z.exe 安装包：
  1. 双击安装程序，按向导选择安装目录
  2. 可勾选创建桌面快捷方式
  3. 从开始菜单或桌面启动「净页 JingYe」
  卸载：系统「应用和功能」中卸载，或开始菜单卸载项

  绿色包与安装版功能相同，仅分发形式不同。
